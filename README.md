# MERN Quiz App

A quiz web application built using the MERN stack.

## Features
- User login & signup
- Quiz selection & attempt
- Score tracking

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## How to Run (Local)
1. Run `npm install` inside `backend` and `frontend`
2. Add your own `.env` file in backend with:
MONGODB_URI=your_connection_string
JWT_SECRET=your_secret
PORT=5000

3. Start backend: `cd backend && node server.js`
4. Start frontend: `cd frontend && npm start`
